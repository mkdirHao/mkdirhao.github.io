# prerare
go develop envirement > 1.18
install zsh 
# start
1.[nali](https://github.com/zu1k/nali)
```bash
go install github.com/zu1k/nali
```
2.zsh

add this in .zshrc 
```zsh
	ping () {
	   /bin/ping $1 $2 $3 | nali
	}
```
